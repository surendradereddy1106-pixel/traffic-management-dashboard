 Traffic Management Dashboard

This project is a web-based Traffic Management Dashboard developed using HTML, CSS, and JavaScript.

## Features
- Traffic data entry for vehicles
- Vehicle count monitoring
- Traffic signal control (North, South, East, West)
- Automatic and manual signal control
- Traffic status display
- Average vehicles per road calculation
- Peak direction detection

## Technologies Used
- HTML
- CSS
- JavaScript

## Output
The dashboard allows users to enter road traffic data, monitor vehicle count, and control traffic signals in real time.
